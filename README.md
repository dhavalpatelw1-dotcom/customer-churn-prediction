# 🚀 Customer Churn Prediction System

## Business Problem

Customer churn directly impacts company revenue. This project predicts whether a customer is likely to leave using Machine Learning.

## Project Highlights

- End-to-End ML Pipeline
- Data Cleaning & EDA
- Feature Engineering
- Random Forest Classifier
- Model Persistence using Joblib
- Real-Time Prediction
- Streamlit Web Application

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

## Model Performance

Accuracy: 78.75%
